# 커맨드 명명 규칙

> 슬래시 커맨드(`.claude/commands/*.md`) 파일명에 적용하는 규칙.

---

## 접두사 규칙

모든 커맨드는 **풀네임**과 **단축명** 두 가지를 함께 만든다.

| 접두사 | 의미 | 형식 | 예시 |
|--------|------|------|------|
| `c_` | 풀네임 (complete) | `c_{동사}_{대상}` | `c_create_project` |
| `s_` | 단축명 (short) | `s_{약어}` | `s_cp` |

- 풀네임과 단축명은 **동일한 .md 내용**을 공유한다 (파일 복사)
- 어느 쪽으로 호출해도 동일하게 동작한다

## 약어 생성 규칙

| 풀네임 동사 | 약어 | 예시 |
|------------|------|------|
| `create` | `c` | `s_cp` = create project |
| `delete` | `d` | `s_dp` = delete project |
| `modify` | `m` | `s_mp` = modify project |
| `projects` (목록) | `prj` | `s_prj` = project list |

- 약어는 **동사 첫 글자 + 대상 첫 글자**를 조합한다
- 대상이 `project`면 `p`, 대상이 길면 2~3글자까지 허용한다

## 특수 커맨드

범용적으로 자주 쓰는 커맨드는 접두사 없이 짧은 이름도 허용한다.

| 커맨드 | 풀네임 | 단축명 | 특수 |
|--------|--------|--------|------|
| 프로젝트 목록 | `c_projects` | `s_prj` | `ll` |

## 커맨드 등록부

| 기능 | 풀네임 | 단축명 | 특수 | 파일 수 |
|------|--------|--------|------|---------|
| 프로젝트 목록 | `/c_projects` | `/s_prj` | `/ll` | 3 |
| 프로젝트 생성 | `/c_create_project` | `/s_cp` | - | 2 |
| 프로젝트 삭제 | `/c_delete_project` | `/s_dp` | - | 2 |
| 프로젝트 변경 | `/c_modify_project` | `/s_mp` | - | 2 |
| 세션 저장 | `/save-session` | - | - | 1 |
| 세션 목록 | `/sessions` | - | - | 1 |
| 세션 삭제 | `/delete-session` | - | - | 1 |
| 세션 병합 | `/merge-sessions` | - | - | 1 |

## 신규 커맨드 추가 시

1. 이 문서의 **커맨드 등록부**에 먼저 등록한다
2. `.claude/commands/`에 풀네임 `.md` 파일을 작성한다
3. 단축명 `.md`를 풀네임 파일 복사로 생성한다
4. 필요하면 특수 이름도 추가한다
