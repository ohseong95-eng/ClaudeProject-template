# Project Config — ClaudeProjects (루트)

> 에이전트가 참조하는 프로젝트 공통 설정값.
> 하드코딩 대신 이 파일의 값을 읽어 사용한다.

---

## 설정값

| 키 | 값 | 설명 |
|----|----|------|
| `max_retry_count` | 10 | 전파·작업 실패 시 최대 재시도 횟수 |
| `backup_dir` | `.claude/_temp_backup` | 임시 백업 저장 경로 |
| `backup_ttl_hours` | 24 | 백업 보존 시간 (시간 초과 시 경고) |

---

## 사용 규칙

- 에이전트는 재시도 횟수, 백업 경로 등을 **하드코딩하지 않는다**
- 작업 전 이 파일을 읽고 해당 설정값을 사용한다
- 설정값 변경 시 이 파일만 수정하면 모든 에이전트에 반영된다
- 각 서브 프로젝트도 동일한 형식의 `.claude/config.md`를 둔다
- 서브 프로젝트 config가 있으면 해당 프로젝트 값 우선, 없으면 루트 config를 사용한다
