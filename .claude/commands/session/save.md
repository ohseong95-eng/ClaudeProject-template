# 세션 저장

현재 사용자와의 대화 내역을 `sessions/` 폴더에 마크다운 파일로 저장합니다.

## 저장 규칙

1. 파일명: `sessions/YYYY-MM-DD-HHMMSS.md` (현재 날짜/시각 기준)
2. 아래 템플릿에 맞게 대화 내용을 요약하여 작성
3. HANDOFF.md와 혼동하지 말 것 — 이 파일은 **사용자와의 대화 기록**용

## 템플릿

```markdown
# 세션 기록: YYYY-MM-DD HH:MM

## 주요 논의 내용
- (대화에서 다룬 주제 요약)

## 결정된 사항
- (이번 세션에서 확정된 결정들)

## 완료된 작업
- (실제로 파일을 생성하거나 수정한 작업들)

## 영향받은 프로젝트
- (이번 세션에서 변경이 적용된 하위 프로젝트 목록)

## 미해결 사항 / 다음에 이어서 할 일
- (완료되지 않은 작업이나 후속 논의 필요 항목)

## 참고 파일
- (이번 세션에서 열람하거나 수정한 주요 파일 목록)
```

## 실행

위 템플릿을 채워 `sessions/YYYY-MM-DD-HHMMSS.md` 파일로 저장하세요.
저장 후 파일 경로를 사용자에게 알려주세요.
