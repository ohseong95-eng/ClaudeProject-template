# 세션 삭제

`/session:list` 목록의 번호를 받아 해당 세션 파일을 삭제합니다.

## 인자

- `$ARGUMENTS` — 삭제할 세션 번호 (쉼표 또는 공백 구분으로 복수 가능)
- 예: `/session:delete 3` 또는 `/session:delete 1,3,5`

## 실행

1. `sessions/` 폴더의 `.md` 파일을 날짜순(오래된 순)으로 정렬하여 번호를 매긴다 (`.gitkeep` 제외)
2. `$ARGUMENTS`에서 번호를 파싱한다
3. 삭제 전 대상 파일 목록을 사용자에게 보여주고 **확인을 받는다**
4. 확인 후 해당 파일을 삭제한다
5. 결과를 출력한다

## 출력 형식

```
## 삭제 대상

| # | 파일명 |
|---|--------|
| 3 | 2026-02-20-abcd1234.md |

삭제하시겠습니까? (삭제는 되돌릴 수 없습니다)
```

확인 후:

```
1개 세션 삭제 완료
- sessions/2026-02-20-abcd1234.md
```

## 안전 규칙

- 번호가 범위를 벗어나면 오류 메시지를 출력하고 중단한다
- **반드시 사용자 확인 후 삭제**한다 — 자동 삭제 금지
