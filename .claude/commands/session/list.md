# 세션 목록

`sessions/` 폴더의 모든 세션 파일을 번호와 함께 조회합니다.

## 실행

1. `sessions/` 폴더의 `.md` 파일을 날짜순(오래된 순)으로 정렬한다
2. 각 파일의 첫 줄(제목), Session ID, Turns 수를 읽는다
3. 아래 형식으로 출력한다

## 출력 형식

```
## 세션 목록 (sessions/)

| # | 파일명 | 세션 시작 | Turns | Session ID |
|---|--------|----------|-------|------------|
| 1 | 2026-02-20-abcd1234.md | 2026-02-20 14:30 | 12 | abcd1234... |
| 2 | 2026-02-20-efgh5678.md | 2026-02-20 16:00 | 5 | efgh5678... |

총 2개 세션, 17 turns
```

## 참고

- `.gitkeep` 파일은 목록에서 제외한다
- 번호는 `/session:delete`와 `/session:merge`에서 사용한다
