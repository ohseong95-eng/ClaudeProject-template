# 공용 문서 관리

`docs/public/` 공유 참조 문서를 조회·추가·삭제·변경한다.
변경 시 CLAUDE.md 테이블과 프로젝트 내 참조 경로를 함께 갱신한다.

## 인자

$ARGUMENTS — 동작 지정 (선택). 생략하면 목록 조회.

- `add` — 문서 추가
- `remove <파일명>` — 문서 삭제
- `rename <기존명> <새이름>` — 문서 이름 변경
- (인자 없음) — 목록 조회

## 실행 절차

### 목록 조회 (인자 없음)

1. `docs/public/*.md` 파일을 Glob으로 수집한다
2. 각 파일의 첫 번째 `#` 제목 행을 읽는다
3. 아래 형식으로 출력한다:
   ```
   # 공용 참조 문서 (docs/public/)

   | # | 문서 | 제목 |
   |---|------|------|
   | 1 | `config-template.md` | Project Config Template |

   총 N개 문서
   ```

### 추가 (`add`)

1. 사용자에게 질문한다:
   - **파일명**: `.md` 확장자 포함 (예: `api-guide.md`)
   - **용도**: 1줄 설명
   - **내용 소스**: 새로 작성 / 기존 파일 이동
2. 파일을 `docs/public/`에 생성 또는 이동한다
3. **연쇄 갱신**을 수행한다:
   - `CLAUDE.md`의 `## 공유 참조 문서` 테이블에 행 추가
4. 결과를 출력한다:
   ```
   ## 추가 완료

   | 항목 | 값 |
   |------|-----|
   | 파일 | docs/public/{파일명} |
   | 갱신 | CLAUDE.md 테이블 행 추가 |
   ```

### 삭제 (`remove <파일명>`)

1. `docs/public/{파일명}` 존재 여부를 확인한다
2. 프로젝트 전체에서 해당 파일 참조를 Grep으로 검색한다 (sessions/ 제외)
3. 참조 목록을 보여주고 사용자에게 삭제 확인을 받는다:
   ```
   ## 삭제 영향 범위

   | # | 파일 | 행 | 참조 내용 |
   |---|------|----|----------|
   | 1 | CLAUDE.md | 70 | `docs/public/xxx.md` |
   | 2 | ... | ... | ... |

   위 참조를 모두 제거하고 문서를 삭제하시겠습니까?
   ```
4. 승인 시 **연쇄 갱신**을 수행한다:
   - `CLAUDE.md`의 `## 공유 참조 문서` 테이블에서 해당 행 삭제
   - 본문 내 `docs/public/{파일명}` 참조를 모두 제거 또는 대체
   - `pending/pending-delete.md`에 등록 후 `/file:delete` 절차를 따른다

### 이름 변경 (`rename <기존명> <새이름>`)

1. `docs/public/{기존명}` 존재 여부를 확인한다
2. 프로젝트 전체에서 `docs/public/{기존명}` 참조를 Grep으로 검색한다 (sessions/ 제외)
3. 변경 계획을 보여주고 사용자에게 확인을 받는다:
   ```
   ## 이름 변경 계획

   | 항목 | 변경 |
   |------|------|
   | 파일 | {기존명} → {새이름} |

   ### 참조 갱신 대상
   | # | 파일 | 행 | 현재 | 변경 후 |
   |---|------|----|------|--------|
   | 1 | CLAUDE.md | 70 | docs/public/{기존명} | docs/public/{새이름} |
   ```
4. 승인 시 **연쇄 갱신**을 수행한다:
   - 파일 이름을 변경한다
   - `CLAUDE.md` 테이블의 파일명을 갱신한다
   - 본문 내 모든 `docs/public/{기존명}` 참조를 `docs/public/{새이름}`으로 교체한다

## 연쇄 갱신 대상

공용 문서 변경 시 반드시 확인·갱신해야 하는 파일:

| 대상 | 위치 | 갱신 내용 |
|------|------|----------|
| CLAUDE.md 테이블 | `## 공유 참조 문서` 섹션 | 행 추가/삭제/파일명 변경 |
| CLAUDE.md 본문 | 전체 | `docs/public/` 참조 경로 |
| docs/project-creation-guide.md | 체크리스트 | `docs/public/` 참조 경로 |
| 각 에이전트/스킬 프롬프트 | `.claude/agents/`, `.claude/skills/` | `docs/public/` 참조 경로 |
