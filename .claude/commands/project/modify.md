# 서브 프로젝트 변경

서브 프로젝트의 레지스트리 정보(역할, 상태 등)를 변경한다.

## 출력 규칙

- 중간 설명 없이 진행 상황과 결과만 출력한다

## 실행 절차

1. 사용자에게 변경할 프로젝트 번호 또는 이름을 확인한다
   - `docs/project-registry.md` 레지스트리를 읽어 목록을 보여준다

2. 변경할 항목을 질문한다 (복수 선택 가능)
   - 역할
   - 입력 소스
   - 출력 대상
   - 상태 (활성 ↔ 사용 금지)

3. 변경 사항을 반영한다
   - `docs/project-registry.md` — 레지스트리

4. 변경 전후 비교 표를 출력한다
