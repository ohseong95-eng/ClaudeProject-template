# 서브 프로젝트 삭제

서브 프로젝트를 레지스트리에서 제거하고 관련 파일을 정리한다.

## 출력 규칙

- 중간 설명 없이 진행 상황과 결과만 출력한다

## 실행 절차

1. 사용자에게 삭제할 프로젝트 번호 또는 이름을 확인한다
   - `docs/project-registry.md` 레지스트리를 읽어 목록을 보여준다

2. 삭제 전 확인 질문을 한다
   - "디렉토리도 함께 삭제할까요?" (레지스트리만 제거 / 디렉토리 포함 삭제)

3. 레지스트리 정리
   - `docs/project-registry.md` — 해당 프로젝트 행 제거

4. 디렉토리 삭제를 선택한 경우 `pending/pending-delete.md`에 기록한다
   - 실제 삭제는 `/file:delete` 커맨드를 통해서만 수행한다

5. 완료 후 결과 요약을 출력한다
