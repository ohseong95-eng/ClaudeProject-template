# 서브 프로젝트 생성

새 서브 프로젝트를 생성하고 레지스트리에 등록한다.

## 출력 규칙

- 중간 설명 없이 진행 상황과 결과만 출력한다

## 실행 절차

1. 사용자에게 아래 정보를 질문한다
   - 프로젝트명 (디렉토리명)
   - 역할 (1~2문장)
   - 에이전트 설계 방식 (capability 기반 / 도메인 전담)

2. `docs/project-creation-guide.md`의 생성 체크리스트를 따라 순서대로 수행한다

3. `docs/project-creation-guide.md`의 필수 디렉토리 구조에 명시된 폴더를 모두 생성한다

4. `docs/project-registry.md` 레지스트리에 신규 프로젝트를 추가한다

5. 완료 후 결과 요약 표를 출력한다
