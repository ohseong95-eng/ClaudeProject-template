# 삭제 대기 목록 조회

`pending/pending-delete.md`에 등록된 삭제 대기 항목을 조회한다.

## 출력 규칙

- 도구 호출 과정, 중간 설명, 부가 대화를 일절 출력하지 않는다
- 결과만 출력한다

## 실행 절차

1. `pending/pending-delete.md`를 읽는다
2. 각 항목의 실제 파일 존재 여부를 확인한다
3. 아래 형식으로 출력한다

## 출력 형식

```
## 삭제 대기 목록

| # | 대상 | 사유 | 존재 |
|---|------|------|------|
| 1 | `some/file.md` | 사유 | O |
| 2 | `some/dir/` | 사유 | X (이미 없음) |

---

총 N개 대기 중 (존재 N / 이미 없음 N)

삭제하려면 `/file:delete <번호>` 또는 `/file:delete all`
```
