# PROFILE — ClaudeProjects (Root)

## 프로젝트 설명

서브 프로젝트들을 통합 관리하는 메타 프로젝트.
프로젝트 생성·삭제·동기화, 명명 규칙 관리, 크로스 프로젝트 오케스트레이션을 담당한다.

## 프로젝트 레지스트리

> 프로젝트 목록은 `docs/project-registry.md`를 참조한다.
> 신규 프로젝트 추가 시 `docs/project-registry.md`를 업데이트한다.

### 레지스트리 상태 규칙

- 상태가 **사용 금지**인 프로젝트는 에이전트 실행, 작업 지시, 데이터 송수신 대상에서 **제외**한다
- 모든 에이전트는 작업 전 레지스트리의 상태를 확인한다
- 사용 금지 프로젝트에 작업을 시도하면 사용자에게 경고하고 중단한다

## 특이사항

- 이 프로젝트 자체는 코드를 실행하지 않음 — 관리·조율 전용
- 서브 프로젝트 작업 시 해당 프로젝트의 `PROFILE.md`와 `.claude/config.md`를 우선 참조
- 서브 프로젝트의 변경 건은 사용자가 직접 명령하지 않는 한 해당 프로젝트의 `todo/`에 추가한다 — 안정성 우선
